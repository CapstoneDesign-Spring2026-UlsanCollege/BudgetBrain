## MVP Features Ready

- User registration and login with JWT
- Password reset via email codes
- Budget creation, editing, deletion
- Expense recording, editing, deletion
- Savings goal creation and progress tracking
- Dashboard with summary cards and analytics charts
- Dark/light theme toggle
- Avatar selection on profile
- Currency conversion widget
- Responsive design
- Premium UI with 3D and liquid effects

## Features Planned

- Bank/card transaction integration
- Receipt scanning with OCR
- AI-powered spending insights
- Monthly financial reports
- Email/push notifications
- Data export (PDF/CSV)
- Multi-language support
- Multi-factor authentication
- PWA support
