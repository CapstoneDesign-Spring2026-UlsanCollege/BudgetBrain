# BudgetBrain - Pitch Issue

## Problem
College students struggle to track their finances across multiple categories. Existing tools are either too complex (QuickBooks) or don't support NPR currency natively.

## Solution
BudgetBrain - a simple, beautiful budget planner that:
- Lets students create budgets per category
- Tracks expenses in real-time
- Visualizes spending with analytics charts
- Supports savings goals
- Works in NPR (Nepalese Rupee)

## Target Users
- College students at Ulsan College
- International students managing living expenses
- Anyone needing a lightweight budget tracker

## Unique Value
- Free and open-source
- NPR currency support by default
- Beautiful 3D/liquid UI
- Works offline with in-memory fallback
- No paid subscriptions or ads

## Success Metrics
- Active users
- Budgets created
- Expenses tracked
- Goals achieved
- User satisfaction

## Tech Stack
- Frontend: React + Vite
- Backend: Express + MongoDB
- Auth: JWT
- Deployment: Vercel for the full-stack app
