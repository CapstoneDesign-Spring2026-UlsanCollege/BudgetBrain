## Detailed Notes

This week was about planning before coding. We drew wireframes on paper first, then created digital versions. The design goal was to make the app look modern and premium without using a CSS framework.

We wanted a sidebar navigation because it gives easy access to all pages. On mobile, the sidebar collapses into a hamburger menu. The dashboard shows a summary of all budgets with color-coded cards.

Color coding was an important decision. Each budget category gets a distinct color so users can quickly scan and identify them. We created a palette of 10+ colors including pink, indigo, teal, rose, coral, lime, sky, amber, emerald, and violet.

The dark/light theme was planned from the start. We use CSS variables so switching themes is just changing variable values. No CSS framework needed.

The 3D card effects and liquid animations were added to make the UI feel premium. These are CSS-only effects, no JavaScript needed for the animations.
