# Week 4 - UI Planning and Wireframes

**Date:** March 23-27, 2026

## What We Did

We planned the user interface this week. We drew wireframes for all the pages and decided on the look and feel of the app. We wanted it to look modern and premium with 3D effects and smooth animations.

## Pages Designed

- Login page
- Registration page
- Dashboard with budget overview
- Budget detail with expenses list
- Savings goals page
- Analytics page with charts
- Profile page
- Settings page

## Design Decisions

- Sidebar navigation for easy access to all pages
- Dark and light theme support using CSS variables
- 3D card effects for budget items
- Color-coded categories for easy recognition
- Mobile responsive layout
