# Week 1 - Project Idea and Team Setup

**Date:** March 2-6, 2026

## What We Did

This week was all about getting started. We formed our team and brainstormed project ideas. After discussing different options, we decided to build BudgetBrain, a budget planning web app for college students.

The main reason we chose this idea is because many students including ourselves struggle to keep track of spending. We wanted something simple but useful.

## Team

- BudgetBrain Development Team
- Ulsan College, Capstone Design - Spring 2026

## Decisions Made

- Tech stack: React + Vite for frontend, Express + MongoDB for backend
- JWT for authentication
- Vanilla CSS (no frameworks)
- GitHub for version control
