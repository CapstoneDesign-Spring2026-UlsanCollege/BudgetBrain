# BudgetBrain Project Overview

## Project Name

**BudgetBrain**

## Project Type

Personal Finance Management Web Application

## Course

Capstone Design — Spring 2026  
Ulsan College, South Korea

## Problem Statement

Many college students, especially international students, struggle to manage their personal finances. Tracking expenses, staying within budgets, and saving money are common challenges. Existing finance apps are either too complex, require bank account connections, or are not tailored to student needs. There is a need for a simple, accessible budget planning tool that works on any device.

## Proposed Solution

BudgetBrain is a full-stack web application that allows users to:

- Create and manage budgets by category
- Record and track expenses
- Set savings goals and monitor progress
- View spending analytics through visual charts
- Manage their profile and preferences

The app uses a clean, modern interface with dark/light theme support and works on both desktop and mobile devices.

## Target Users

- College and university students
- Young professionals managing personal finances
- Anyone looking for a simple budget tracking tool

## Main Features

### Implemented (MVP)

- User registration and login with JWT authentication
- Password reset via email reset codes
- Budget creation, editing, and deletion by category
- Expense recording, editing, and deletion within budgets
- Savings goal creation with progress tracking
- Spending analytics with Recharts visualizations
- Dark and light theme toggle
- Avatar selection on user profile
- Currency conversion widget (NPR default)
- Responsive design for mobile and desktop
- Premium UI with 3D and liquid glass effects

### Planned

- Bank/card transaction integration
- Receipt scanning with OCR
- AI-powered spending insights
- Monthly financial report generation
- Email and push notifications
- Data export (PDF/CSV)
- Multi-language support
