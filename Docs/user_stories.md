# BudgetBrain - User Stories

## Authentication
- As a user, I want to register an account so I can save my data
- As a user, I want to login so I can access my budgets
- As a user, I want to see distinct error messages so I know if my email or password is wrong
- As a user, I want to register without being auto-logged in so I can verify my credentials manually

## Budget Management
- As a user, I want to create budgets so I can plan my spending
- As a user, I want to edit my budgets so I can adjust amounts
- As a user, I want to delete budgets I no longer need
- As a user, I want to see my budgets color-coded so I can distinguish them easily

## Expense Tracking
- As a user, I want to add expenses to a budget so I can track spending
- As a user, I want to see all my expenses in a list so I can review them
- As a user, I want to edit expense amounts so I can correct mistakes
- As a user, I want to delete expenses I added by mistake

## Savings Goals
- As a user, I want to create savings goals so I can save for specific items
- As a user, I want to add money to my goals so I can track progress
- As a user, I want to see my progress visually so I stay motivated
- As a user, I want to set deadlines so I can plan my savings timeline

## Analytics
- As a user, I want to see spending charts so I understand my habits
- As a user, I want to see budget vs actual so I know if I'm overspending
- As a user, I want to see monthly trends so I can compare periods

## Settings
- As a user, I want to switch between dark and light themes
- As a user, I want to choose my currency so amounts display correctly
- As a user, I want to select an avatar for my profile
- As a user, I want to edit my display name

## UI/UX
- As a user, I want a responsive design so it works on mobile
- As a user, I want smooth animations so the app feels premium
- As a user, I want clear navigation so I can find features easily
