# BudgetBrain User Guide

## Getting Started

1. Go to https://budget-brain-puce.vercel.app
2. Click Register and create an account
3. Login with your email and password

## Creating a Budget

1. From the Dashboard, click "Add New Budget"
2. Enter a name (like "Food" or "Rent")
3. Set the budget amount
4. Pick a color
5. Click Save

## Adding Expenses

1. Click on a budget to open it
2. Click "Add Expense"
3. Enter the expense name and amount
4. Select the date
5. Click Save

## Setting Savings Goals

1. Go to the Goals page from the sidebar
2. Click "Create New Goal"
3. Name your goal (like "New Laptop")
4. Set the target amount
5. Add a deadline (optional)
6. Click Save
7. Use "+ Add Savings" to add money to your goal

## Viewing Analytics

1. Go to the Analytics page
2. See a pie chart of your spending by category
3. See a bar chart of monthly spending trends

## Changing Settings

1. Go to Settings from the sidebar
2. Toggle between Dark and Light theme
3. Change your currency preference
4. Toggle notifications on/off

## Editing Profile

1. Go to Profile
2. Select an avatar
3. Edit your display name
4. Click Save Changes

## Tips

- Use color-coded budgets to organize your spending
- Check the Dashboard regularly to stay on track
- Set savings goals for things you want to buy
- Use dark mode at night to reduce eye strain
