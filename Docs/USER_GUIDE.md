# BudgetBrain User Guide

## Getting Started

1. Go to https://budgetbrain.vercel.app/
2. Click Register and create an account (name, email, password)
3. Login with your email and password
4. You'll land on the Dashboard — your command center

## Creating a Budget

1. From the Dashboard, click "Add New Budget"
2. Enter a name (like "Food" or "Rent")
3. Set the budget amount
4. Pick a color
5. Click Save
6. The budget appears on your dashboard with a progress bar

## Adding Expenses

1. Click on a budget card to open it
2. Click "Add Expense"
3. Enter the expense name and amount
4. Select the date
5. Click Save
6. The expense appears in your transaction list

## Setting Savings Goals

1. Go to the Goals page from the sidebar
2. Click "Create New Goal"
3. Name your goal (like "New Laptop")
4. Set the target amount
5. Add a deadline (optional)
6. Click Save
7. Use "+ Add Savings" to add money to your goal
8. Watch your progress bar grow

## Viewing Analytics

1. Go to the Analytics page from the sidebar
2. Pie chart: spending breakdown by budget category
3. Bar chart: monthly spending trends over time
4. Hover over chart elements for detailed values

## Scanning Receipts

1. Go to the Receipts page
2. Upload a receipt image (JPG, PNG)
3. The app extracts text using OCR
4. Review and use the extracted information

## Changing Settings

1. Go to Settings from the sidebar
2. Toggle between Dark and Light theme
3. Change your currency preference (affects dashboard summaries)
4. Settings persist across sessions

## Editing Profile

1. Go to Profile from the sidebar
2. Select an emoji avatar
3. Edit your display name
4. Click Save Changes

## Password Reset

1. On the login page, click "Forgot Password?"
2. Enter your registered email
3. Check your email for a reset code
4. Enter the code and your new password
5. Login with your new password

## Troubleshooting

### Login Issues
- Verify your email and password are correct
- Use "Forgot Password?" to reset if needed
- Clear browser cache and try again

### Data Not Loading
- Check your internet connection
- Refresh the page
- The app needs a connection to the backend API

### Theme Not Saving
- Ensure localStorage is enabled in your browser
- Try clearing site data and re-setting your preference

### Receipt Scan Fails
- Ensure the image is clear and well-lit
- Try a different image format (JPG recommended)
- The OCR service may be unavailable — try again later

## Tips

- Use color-coded budgets to organize your spending
- Check the Dashboard regularly to stay on track
- Set savings goals for things you want to buy
- Use dark mode at night to reduce eye strain
- Scan receipts immediately after purchases
